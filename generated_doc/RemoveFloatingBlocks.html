Search for floating voxels blocks in a box of radius Radius centered in Position, and spawn ClassToSpawn for each of those blocks

Inputs

In
Exec

World
Voxel World Object Reference

Voxel world
Class to Spawn
Voxel Part Class Reference

The class of the spawned actors
Position
Vector

Position in world space
Radius
Float

Radius of the box in voxel space (-Radius < X < Radius ...)

Outputs

Out
Exec

Spawned Actors
Array of Voxel Part Object References
