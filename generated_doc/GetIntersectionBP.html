Get the intersection using voxel data. Doesn't depend on LOD. Useful for short distances, but costful for big ones

Inputs

In
Exec

Target
Voxel World Object Reference

Start
Int Vector Structure (by ref)

The start of the raycast. The start and the end must have only one coordinate not in common
End
Int Vector Structure (by ref)

The end of the raycast. The start and the end must have only one coordinate not in common

Outputs

Out
Exec

Global Position
Vector

Voxel Position
Int Vector Structure

Return Value
Boolean

GlobalPosition The world position of the intersection if found
