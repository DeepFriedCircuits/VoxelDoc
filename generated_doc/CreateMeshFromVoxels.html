Create a mesh from given values

Inputs

In
Exec

Size
Int Vector Structure (by ref)

Values
Array of Floats

Value array. ValueXYZ = Values[X + Size.X * Y + Size.X * Size.Y * Z]. Can be empty
Materials
Array of Voxel Material Structures

Materials array. MaterialXYZ = Materials[X + Size.X * Y + Size.X * Size.Y * Z]. Can be empty
Voxel Size
Float

The size of a voxel
Material
Material Interface Object Reference

The material to set
Parent
Object Object Reference

The parent of the mesh to be created

Outputs

Out
Exec

Mesh
Voxel Procedural Mesh Component Object Reference
