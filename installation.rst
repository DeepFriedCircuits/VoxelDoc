Installation
============

Manual installation
-------------------

* In your game’s root directory, create a folder named Plugins
* Copy the Voxel folder into it. You should have something like::

    MyProject
    ├── Content
    └── Plugins
        └── Voxel
            └── Voxel.uplugin
